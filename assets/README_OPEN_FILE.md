## ⚠️ 如果打不开？

### 情况1：显示"无法访问此页面"
**说明：** 你在远程环境（云端IDE）

**解决方法：**
1. 如果你使用 VS Code Remote
   - 点击左侧"端口"标签
   - 找到端口 8080
   - 点击"在浏览器中打开"

2. 如果你使用 GitHub Codespaces
   - 等待VS Code自动检测端口
   - 点击弹出的"在浏览器中打开"

---

### 情况2：显示搜索结果
**说明：** 浏览器把地址当成了搜索关键词

**解决方法：**
- 使用 `http://` 开头的完整地址
- 不要输入相对路径

---

## 📱 手机访问

如果想在手机上访问：

1. 确保手机和电脑在同一WiFi网络
2. 查看电脑的局域网IP：
   - Windows: `ipconfig` → 找到IPv4地址
   - Mac/Linux: `ifconfig` 或 `ip addr`
3. 在手机浏览器打开：
   ```
   http://[你的电脑IP]:8080/hanna_chat.html
   ```
   例如：`http://192.168.1.100:8080/hanna_chat.html`
