### 界面功能
- 微信风格顶部导航栏

## 📝 使用说明

### 1. 发送消息
- 在输入框中输入文字
- 点击"发送"按钮或按Enter键发送
- 消息会立即显示在右侧（绿色气泡）

### 2. 查看回复
- 韩娜会显示"正在输入..."动画
- 1-2秒后显示回复（左侧白色气泡）
- 当前使用模拟回复

## 🔧 技术信息

### 服务架构
```
浏览器 → loca.lt隧道 → HTTP服务器(8080) → 静态HTML文件
```

### 使用工具
- **HTTP服务器**：Python SimpleHTTPServer
- **隧道工具**：localtunnel
- **文件类型**：纯静态HTML + CSS + JavaScript

## 💡 注意事项

1. **首次访问**：可能需要等待1-2秒加载
2. **网络要求**：需要稳定的互联网连接
3. **模拟回复**：当前使用模拟回复，连接真实后端需要额外配置
4. **服务持续运行**：隧道和服务器会持续运行，可随时访问
