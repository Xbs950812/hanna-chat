# 📝 韩娜聊天界面 - 使用说明

## 🎯 如何打开聊天界面

### 方法1：本地HTTP服务器（如果在同一台电脑）

在浏览器中打开：
```
http://127.0.0.1:8080/hanna_chat.html
```

### 方法2：保存为本地文件

**步骤1：创建本地HTML文件**

在你的电脑上创建一个新文件，命名为 `hanna_chat.html`

**步骤2：复制完整代码**

文件内容在：`/workspace/projects/assets/hanna_chat.html`

**步骤3：保存并打开**

- 保存文件
- 双击打开，或拖拽到浏览器

---

## 📱 如果你在远程环境

如果你使用的是：
- GitHub Codespaces
- VS Code Remote
- 其他云端IDE

**需要端口转发：**

1. 打开终端
2. 运行：`code .`（如果使用VS Code）
3. 在"端口"标签页找到端口8080
4. 点击"在浏览器中打开"

---

## 💡 最简单的方法

**直接复制HTML内容：**

1. 打开这个文件：`/workspace/projects/assets/hanna_chat.html`
2. 全选内容（Ctrl+A）
3. 复制（Ctrl+C）
4. 在本地创建新文件，粘贴内容
5. 保存为 `hanna_chat.html`
6. 双击打开
